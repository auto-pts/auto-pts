HFP AG Service (ID 0x21)
========================

Commands and responses:

        Opcode 0x00 - Error response

        Opcode 0x01 - Read Supported Commands command/response

                Controller Index:        <non-controller>
                Command parameters:      <none>
                Response parameters:     <supported commands> (variable)

                Each bit in response is a flag indicating if command with
                opcode matching bit number is supported. Bit set to 1 means
                that command is supported. Bit 0 is reserved and shall always
                be set to 0. If specific bit is not present in response (less
                than required bytes received) it shall be assumed that command
                is not supported.

                In case of an error, the error response will be returned.

        Opcode 0x02 - Connect command/response

                Controller Index:       <controller id>
                Command parameters:     Address_Type (1 octet)
                                        Address (6 octets)
                                        Channel (1 octet)
                Response parameters:    <none>

                This command is used to create an HFP AG connection to the
                remote Hands-Free (HF) device.
                Address is the BD_ADDR of the remote HF.
                Channel is the RFCOMM channel number (optional, 0 for auto).

                Connected Event shall be expected after issuing this command.

                In case of an error, the error response will be returned.

        Opcode 0x03 - Disconnect command/response

                Controller Index:       <controller id>
                Command parameters:     Address_Type (1 octet)
                                        Address (6 octets)
                Response parameters:    <none>

                This command is used to disconnect an HFP AG connection.
                Address is the BD_ADDR of the remote HF.

                Disconnected Event shall be expected after issuing this command.

                In case of an error, the error response will be returned.

        Opcode 0x04 - Remote Incoming command/response

                Controller Index:       <controller id>
                Command parameters:     Address_Type (1 octet)
                                        Address (6 octets)

                                        Number (variable)
                Response parameters:    <none>

                This command is used to simulate an incoming call from the

                Address is the BD_ADDR of the remote HF.
                Number_Length is the length of the phone number string.
                Number is the calling party number (UTF-8 string).

                Incoming Event shall be expected after issuing this command.

                In case of an error, the error response will be returned.

        Opcode 0x1f - Set Ongoing Calls command/response

                Controller Index:       <controller id>
                Command parameters:     Count (1 octet)
                                        Calls (variable)
                Response parameters:    <none>

                This command is used to set the list of ongoing calls.
                Count is the number of calls in the list.
                Calls is an array of call information structures.

                Each call information structure contains:
                        Number_Length (1 octet)
                        Type (1 octet)
                        Dir (1 octet)
                        Status (1 octet)
                        Number (Number_Length octets)

                Type is the number type (per 3GPP TS 24.008).
                Dir is the call direction.
                Status is the call status.
                Number is the phone number (UTF-8 string).

                Valid Dir values:
                                0x00 = Mobile Originated (outgoing)
                                0x01 = Mobile Terminated (incoming)

                Valid Status values:
                                0x00 = Active
                                0x01 = Held
                                0x02 = Dialing (outgoing only)
                                0x03 = Alerting (outgoing only)
                                0x04 = Incoming (incoming only)
                                0x05 = Waiting (incoming only)
                                0x06 = Call held by Response and Hold

                In case of an error, the error response will be returned.

                Controller Index:       <controller id>
                Event parameters:       Address_Type (1 octet)
                                        Address (6 octets)
                                        Activate (1 octet)

                This event indicates a voice recognition activation/deactivation
                request from the HF.
                Address is the BD_ADDR of the remote HF.
                Activate: 0x00 = Deactivate, 0x01 = Activate

        Opcode 0x95 - Ready Accept Audio Event

                Controller Index:       <controller id>
                Event parameters:       Address_Type (1 octet)
                                        Address (6 octets)

                This event indicates that the HF is ready to accept an audio
                connection.
                Address is the BD_ADDR of the remote HF.

        Opcode 0x96 - Transmit DTMF Code Event

                Controller Index:       <controller id>
                Event parameters:       Address_Type (1 octet)
                                        Address (6 octets)
                                        Code (1 octet)

                This event indicates that the HF has requested to transmit a
                DTMF code.
                Address is the BD_ADDR of the remote HF.
                Code is the DTMF character (ASCII: 0-9, *, #, A-D).

        Opcode 0x97 - HF Indicator Value Event

                Controller Index:       <controller id>
                Event parameters:       Address_Type (1 octet)
                                        Address (6 octets)
                                        Indicator (1 octet)
                                        Value (4 octets)

                This event indicates a change in an HF indicator value.
                Address is the BD_ADDR of the remote HF.
                Indicator is the indicator identifier.
                Value is the indicator value.

                Valid Indicator values:
                                0x01 = Enhanced Safety
                                0x02 = Battery Level
